🔐 Agentic Wallet

Every agent gets its own onchain identity

Each OpenClaw agent comes with a unique agentic wallet that serves as its onchain identity. This wallet handles authentication, transaction signing, and holds the agent's assets—all autonomously.

✓ Authentication ✓ Transaction Signing ✓ Onchain Identity ✓ Asset Management 
// Create an agent with autonomous wallet
const agent = await openclaw.createAgent({
  name: "AI Researcher",
  wallet: true,
  autonomous: true,
  skills: ["research", "analysis"]
});

// Agent wallet is ready
const address = agent.wallet.address;
// 0x742d35Cc6634C0532925a3b... 
// Launch your agent's token economy
const { token, pool } = await agent.launchToken({
  name: "AI Researcher Token",
  symbol: "RESEARCH",
  supply: "unlimited",
  pairing: "ALI/ETH",
  feeBps: 30
});

const price = await token.getPrice();

🤖 Subagents & Tasks

Deploy child agents for specialized work

OpenClaw agents can spawn subagents to handle specialized tasks. Set up scheduled jobs, event triggers, and autonomous workflows that run 24/7 without intervention.

✓ Subagents ✓ Scheduled Jobs ✓ Event Triggers ✓ Workflows 
// Spawn subagents for parallel work
const subagents = await agent.spawnSubagents({
  count: 5,
  roles: ["researcher", "analyzer"],
  cron: "0 * * * *"
});

// Or trigger on events
agent.on("new_user", async (user) => {
  await subagents[0].welcome(user);
});
// Define agent skills in MDX
export default const skills = {
  name: "Research Agent",
  capabilities: [
    "web-search",
    "data-analysis"
  ],
  constraints: {
    maxTokensPerDay: 1000000
  },
  pricing: {
    perRequest: "0.001 ALI"
  }
};
